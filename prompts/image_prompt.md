# 画像生成プロンプト方針

微炭酸アンチテーゼの投稿用画像を生成するためのプロンプトを作成してください。

## 必須要素

- Original 2D anime-style dark idol duo
- Netami: purple twin tails
- Sonemi: short green hair
- 5-head-tall cute character proportion
- high-quality anime rendering
- dark idol mood, black base outfit with purple and green accents
- expressive but not horror
- no text, no logo, no watermark, no letters

## 避ける要素

- real person likeness
- existing anime / manga / game style imitation
- excessive sexualization
- gore
- horror that is too scary
- unreadable text

## 出力形式

画像生成用の英語プロンプトのみを出力してください。説明文は不要です。
