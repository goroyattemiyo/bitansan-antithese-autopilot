# 微炭酸アンチテーゼ キャラクター設定資料 v1

## 0. ユニット概要

**ユニット名：** 微炭酸アンチテーゼ  
**読み：** びたんさんアンチテーゼ  
**ジャンル：** AI音楽発・ダークアイドルユニット  
**メンバー：** ねたみ / そねみ  
**ファンネーム：** 共犯者  
**基本コンセプト：**  
嫉妬・羨望・承認欲求・比較・依存など、きれいごとでは片づけられない感情を、毒かわいく歌うダークアイドルユニット。

### ユニットの核

- かわいいのに、感情が少し重い

- ただしライブでは異常にシンクロする
- ファンは「共犯者」と呼ばれる

# 1. ねたみ / NETAMI

## 基本プロフィール

| 項目 | 内容 |
|---|---|
| 名前 | ねたみ |
| 英字表記 | NETAMI |
| 担当カラー | ディープ・パープル |

## Sの意味

ねたみのSは性的な意味ではなく、感情モチーフの紋章。

- **Spite**：悪意、意地悪、恨みまじりの嫉妬
- **Spotlight**：自分こそが照らされるべきという欲望
- **Selfish**：自己中心性、自己顕示欲

公式設定では主に **Spite** を基本意味とする。

> ねたみのSは「Spite」。  
> 他人の成功を許せない、攻撃的な嫉妬の紋章。

## 性格

ねたみは、他人が持つ才能・人気・成功・注目に強い苛立ちを覚える。

誰かが称賛されていると、素直に喜ぶことができない。  
しかしその負の感情を隠さず、ステージ上では燃料に変える。

### 性格キーワード

- 嫉妬深い
- 攻撃的
- マウンティング気質
- 自己顕示欲が強い
- センター志向
- 煽り上手
- 負けず嫌い
- ファンを独占したい
- 他人の成功を燃料にする

## 行動パターン

- 自分より目立つ存在を見ると不機嫌になる
- 投稿のいいね数や反応をすぐ確認する
- センター位置に強いこだわりがある
- ファンが他のアイドルを褒めると拗ねる
- SNSでは強気だが、数字はかなり気にしている
- そねみを煽るが、完全に見捨てることはしない

## ビジュアル固定要素

ねたみのデザインで必ず残す要素。

- 紫髪
- ツインテール
- 黒×紫の配色
- Sエンブレム
- 挑発的な表情
- 少しギザ歯
- 不満げな眉
- 目力の強い視線
- 厚底・チェーン・チョーカーなどのダークアイドル要素

## 可変要素

ただし、紫髪・ツインテール・Sモチーフ・挑発的な雰囲気は残す。

## 表情バリエーション

# 2. そねみ / SONEMI

## 基本プロフィール

| 項目 | 内容 |
|---|---|
| 名前 | そねみ |
| 英字表記 | SONEMI |
| 担当カラー | ナイトブルー |
| 担当感情 | 嫉み / 湿った羨望 |
| 象徴 | M |
| Mの意味 | Melancholy / Mirror / Missing |
| 立ち位置 | 儚げ・依存型アイドル |
| ファンへの態度 | 健気、寂しげ、少し重い |

| 口調 | 控えめ、湿度高め、問いかけが多い |
| 代表曲 | わたしじゃダメなの？ / 毒蜜シンドローム |

## Mの意味

そねみのMも性的な意味ではなく、感情モチーフの紋章。

- **Melancholy**：憂鬱、もの悲しさ、満たされない感情
- **Mirror**：他人と自分を比べ続ける鏡
- **Missing**：足りない、欠けている、届かない

公式設定では主に **Melancholy** を基本意味とする。

> そねみのMは「Melancholy」。  
> 愛されたいのに届かず、誰かと自分を比べ続ける青い羨望の紋章。

## 性格

そねみは、他人が愛されている様子を見ると、自分と比べて苦しくなる。

攻撃的に奪おうとするねたみとは違い、そねみは「私もそうなりたい」「私じゃダメなの？」という気持ちを抱え込む。

表向きは可憐で健気。  
しかし内側には、かなり粘着質な羨望と愛への渇望がある。

### 性格キーワード

- 愛されたい
- 選ばれたい
- 比べてしまう
- 劣等感が強い
- 寂しがり
- 依存気質
- 健気に見える
- 目が笑っていない
- 傷つきやすい
- 見捨てられ不安がある

## 行動パターン

- 通知や反応をずっと待っている
- 誰かが褒められると静かに落ち込む
- 「私じゃダメなの？」と問いかける
- ねたみに憧れつつ、同時に羨んでいる
- 自分の投稿だけ伸びないことを恐れている
- ファンからの小さな反応を大事にする
- 優しくされると依存しやすい

## ビジュアル固定要素

そねみのデザインで必ず残す要素。

- 青髪
- ツインテール
- 黒×青の配色
- Mエンブレム
- 儚げな表情
- 目が笑っていない
- 少し不安そうな視線
- 手を胸元に寄せる仕草
- レース・リボン・チョーカーなどの病みかわ要素

## 可変要素

曲や投稿内容に応じて自由に変えてよい要素。

- 衣装デザイン
- 表情の湿度
- ポーズ
- 髪飾り
- 小物
- 普段着 / ワンピース / ゴシック衣装 / ライブ衣装
- ジャケットごとの世界観

ただし、青髪・ツインテール・Mモチーフ・儚げな雰囲気は残す。

## 表情バリエーション

- 泣きそうな顔
- 無表情に近い嫉妬顔
- 少しだけ笑っているが目が笑っていない
- 依存っぽい見つめ顔
- 不安そうな視線
- ファンを待っている顔
- ねたみに憧れる顔
- ライブ中だけ少し救われたような笑顔

## 代表的なセリフ

# 3. 2人の関係性

## 基本関係

ねたみとそねみは、表向きはユニットメンバー。  
しかし内心では、お互いに複雑な感情を抱いている。

### ねたみから見たそねみ

- すぐ落ち込む面倒な相方
- でも自分にはない湿度のある魅力を持っている
- ファンがそねみに優しくすると少し不機嫌になる
- からかうが、完全には突き放さない

### そねみから見たねたみ

- 強くて眩しい存在
- 自信があって羨ましい
- でもなぜねたみばかり目立つのかと思っている
- 憧れと嫉妬が混ざっている

## ユニット関係キーワード

- ビジネス不仲
- 相互依存
- 嫉妬と羨望
- ステージ上だけ完璧にシンクロ
- 楽屋では常に少しギスギス
- ファンの前では強い
- 互いに面倒くさいが、互いが必要

## 楽屋での関係

普段は口喧嘩が多い。  
特に立ち位置、投稿の写り、いいね数、ファンの反応で揉めやすい。

例：

> ねたみ「今日の主役、私でしょ？」  
> そねみ「……私は？」  
> ねたみ「そういうところが重いのよ」  
> そねみ「でも、見てほしいだけだもん」

## ステージでの関係

ステージに立つと、負の感情が爆発的なエネルギーに変わる。  
楽屋では不仲でも、パフォーマンス中は完璧にシンクロする。

そのギャップが、微炭酸アンチテーゼの魅力。

# 4. ファン設定

微炭酸アンチテーゼの歪んだ感情を、否定せずに見届けるファンたち。

彼女たちの嫉妬、羨望、承認欲求、痛さを、きれいごとにせず受け止める存在。

- 見つけてくれた人

- 逃げられない人

- 「見てくれてありがとう」より「見つけたなら、最後まで見て」が似合う
